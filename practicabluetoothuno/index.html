"Práctica con Bluetooth"

Iker López Bonilla
Ing. Mecatrónica
Oscar Emanuelle Garrido Javalera
Ing. Mecatrónica
Fecha: Octubre 18, 2024
Componentes Electrónica

- Resumen -

En esta práctica aprendimos a usar Bluetooth para crear aplicaciones móviles. Usamos un celular y un Arduino conectado mediante el módulo HC 05 para controlar componentes.

La práctica fue realizada con la orientación de nuestro profesor Huber Girón Nieto.

- Introducción -

Nuestro profesor nos enseñó a crear aplicaciones con MIT App Inventor, una plataforma que permite diseñar apps para móviles. Esta plataforma se utiliza para programar por medio de bloques, los cuales indican la accion que cada botón manda al arduino.

- Materiales -

Los materiales utilizados para la práctica fueron los siguientes:

Un celular para usar la aplicación, una computadora para crear la aplicación, cables para las conexiones, un Arduino para controlar los dispositivos y un módulo HC-05 para la señal Bluetooth.

- Desarrollo -

Después de aprender a usar MIT App Inventor, creamos un circuito simple con un Arduino y un LED para poder encenderlo y apagarlo desde el celular. Luego hicimos otra aplicacion, la cual simulaba una lampara que prendía y apagaba.

Después, escribimos un código de arduino, el cual sevía para traducir las señales de la aplicacion hacia el circuito de la placa.

- Simulación

Después de aprender a usar MIT App Inventor, primero, nos aseguramos de que la aplicación pudiera encender y apagar el LED desde el celular sin problemas, lo cual realizó sin problemas.

El arduino prendió y apagó tal como se esperaba.

Luego, probamos la app de la lámpara, donde los botones cambiaban la pantalla a blanco y negro, simulando que encendíamos o apagábamos una lámpara.

Igualmente, funcionó como se solicitó, sin retrasos o fallo alguno.

Diseño

La aplicación era muy simple, pero contaba con los botones de encendido y apagado de forma clara y directa.

El cableado no estorbaba, por lo que era sencillo de entender y replicar.

Construcción

La aplicacion fue bastante sencilla de realizar, pues los bloques no contaban con indicaciones muy complicadas. Igualmente, el código fue bastante sencillo de escribir.

- Resultados -

Las dos aplicaciones sirvieron como se solicitó, lo que se demostró al cumplir con las indicaciones pedidas.

El arduino fue conectado a la computadora sin ningún probelma, y mostró la señal sin retrasos o cortos.

- Conclusiones -

Aprendimos a conectar estas aplicaciones con el Arduino usando el módulo HC 05, lo que nos permitió controlar diferentes dispositivos de manera inalámbrica, como encender luces o mover motores.

En resumen, la práctica nos dio las bases para proyectos futuros, los cuales usarían más usos con el bluetooth.
